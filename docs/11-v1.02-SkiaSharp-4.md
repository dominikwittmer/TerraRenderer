# TerraRenderer v1.02 – SkiaSharp 4 API update

The AssetBuilder drawing code now uses the current SkiaSharp 4 APIs.

- `SKSamplingOptions` with the Mitchell cubic resampler replaces `SKFilterQuality`.
- `SKFont` owns text size instead of `SKPaint.TextSize`.
- `DrawText` uses the string/font/alignment overload.
- Obsolete `DrawBitmap` overloads were removed.
