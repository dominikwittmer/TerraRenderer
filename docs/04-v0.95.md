# v0.95 rendering pipeline

1. Orthographic projection
2. Material sample (albedo, emission, water, ice, height)
3. Relief normal calculation
4. Diffuse + hemisphere lighting
5. Ocean specular + Fresnel
6. Night-light emission
7. Surface atmospheric haze
8. Linear-space filmic tone mapping
9. PNG export

Optional grayscale maps are sampled equirectangularly. When not configured, material properties are inferred from the supplied day texture so the project remains immediately runnable.
