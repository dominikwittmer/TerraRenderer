# TerraRenderer v0.96 – Watchface Match

## Goal

Match the visual language of the Huawei Earth watchface at 466 × 466 px rather than maximizing physical realism.

## Changes

- Enlarged and repositioned both globe layouts.
- Rebuilt the atmosphere as a thin white-blue inner rim.
- Broadened the day/night transition.
- Reduced night-side atmosphere and polar dominance.
- Deepened ocean blue and reduced cyan.
- Added subtle vegetation and desert grading.
- Retuned ocean Fresnel/specular response.
- Made city lights warmer with more variation between cores and halos.
- Updated the default configuration for the Watchface profile.

## Validation

Reference times remain 00:00, 06:00, 12:00 and 18:00 UTC for both layouts.
